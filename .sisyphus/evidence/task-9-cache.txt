Surface area (matches spec)

type ModelPriceCache struct {
    mu    sync.RWMutex
    items map[string]*model.ModelPrice
}

func NewModelPriceCache(db *gorm.DB) (*ModelPriceCache, error)
func (c *ModelPriceCache) Get(modelID string) (*model.ModelPrice, bool)
func (c *ModelPriceCache) Invalidate(db *gorm.DB) error
func (c *ModelPriceCache) List() []*model.ModelPrice

Design notes
------------
- Keys are normalized via strings.ToLower(strings.TrimSpace(modelID)) so
  lookups tolerate casing/whitespace drift from upstream model IDs.
- Invalidate builds the replacement map outside the critical section and
  swaps it under Lock, so readers never observe a torn snapshot.
- Invalidate on nil db returns gorm.ErrInvalidDB and preserves the prior
  snapshot.
- No goroutines, no timers — refresh is strictly manual, driven by admin
  handlers (per design).

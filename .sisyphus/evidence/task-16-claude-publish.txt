Task 16 — executor/claude.go usage.PublishRecord integration
============================================================

Scope
-----
Modify ONLY executor/claude.go. Parse Claude usage on every request
(both Execute non-stream path and ExecuteStream SSE path) and fire-
and-forget a cliproxyusage.Record via PublishRecord so the SDK
default usage manager routes it to the registered UsagePlugin
(sdk/usage.go). The plugin reads SettleCtx off ctx to decide
whether to settle.


Changes applied
---------------
executor/claude.go
  + import cliproxyusage "github.com/router-for-me/CLIProxyAPI/v7/sdk/cliproxy/usage"

  Execute:
    - capture startedAt at entry
    - ParseClaudeUsage(payload) after body read
    - expose parsed tokens under Response.Metadata["usage"]
      (non-SDK callers keep their existing compatibility surface)
    - call e.publishUsage(...) on BOTH success and failure paths

  ExecuteStream:
    - capture startedAt at entry
    - on pre-stream error: publish a failed record immediately
    - during streaming: accumulate raw SSE body in a bytes.Buffer
    - on ctx cancel / read error / EOF: publish via publishStreamUsage
      which runs parseClaudeStreamUsage over the accumulated bytes

  New helpers (claude-local, mirroring gemini/codex/vertex pattern):
    claudeRequestedModel(req, opts) → upstream model string
    (*ClaudeExecutor).publishUsage(...)          — assembles & fires cliproxyusage.PublishRecord
    (*ClaudeExecutor).publishStreamUsage(...)    — parses SSE body then delegates
    parseClaudeStreamUsage(body) (UsageTokens, bool)
      - fast path: plain JSON body via ParseClaudeUsage
      - SSE path: scan every `data: {json}` event, merge
        (max-of-input, max-of-output, max-of-reasoning,
         max-of-cached) because message_start carries input_tokens
        and message_delta carries output_tokens / cache totals.
    truncateClaudeFailureBody(payload) string    — 4 KiB cap for Fail.Body


Record field population
-----------------------
  Provider    = "claude"                 (providerClaude const)
  Model       = claudeRequestedModel()
  Alias       = cliproxyusage.RequestedModelAliasFromContext(ctx)
  Source      = "claude"
  RequestedAt = startedAt
  Latency     = time.Since(startedAt)
  Failed      = statusCode >= 400 || streamErr != nil
  AuthID      = auth.ID
  AuthIndex   = auth.Index
  AuthType    = auth.Provider
  Detail      = { Input, Output, Reasoning, Cached, TotalTokens=Input+Output }
  Fail        = { StatusCode, Body (truncated to 4 KiB) } — only when failed

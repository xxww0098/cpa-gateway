Task 17 — executor/gemini.go 调 usage.PublishRecord
===================================================

File modified
-------------
- executor/gemini.go
  * add import `cliproxyusage "github.com/router-for-me/CLIProxyAPI/v7/sdk/cliproxy/usage"`
  * Execute: capture startedAt, parse ParseGeminiUsage(payload) on the full
    response body, stash tokens in Response.Metadata["usage"] (back-compat),
    call publishUsage(ctx, auth, tokens, ok, failed, status, payload, startedAt).
  * ExecuteStream: capture startedAt. On upstream 4xx/5xx (pre-goroutine)
    read the body and publishUsage({}, false, true, ...). In the streaming
    goroutine, accumulate every chunk into bytes.Buffer; on EOF/error/ctx.Done
    call publishStreamUsage(ctx, auth, accumulator.Bytes(), ...) which scans
    for the final `usageMetadata`-bearing chunk and delegates to publishUsage.
  * new unexported helpers:
      - publishUsage(ctx, auth, tokens, parsed, failed, status, payload, startedAt)
      - publishStreamUsage(ctx, auth, body, failed, status, startedAt)
      - parseGeminiStreamUsage(body) — handles 3 framings:
          1. plain JSON (fast path via ParseGeminiUsage)
          2. SSE `data: {json}` events (this executor sets alt=sse)
          3. `\n\n`-separated JSON chunks (non-SSE streamGenerateContent)
        Keeps the last successful parse — matches the spec note "the final
        chunks contain usageMetadata".
      - truncateGeminiFailureBody(payload) — 4 KiB cap on Fail.Body.

  Token mapping (from ParseGeminiUsage) into cliproxyusage.Detail:
    promptTokenCount        → InputTokens
    candidatesTokenCount    → OutputTokens
    thoughtsTokenCount      → ReasoningTokens   (Gemini "thinking" tokens)
    cachedContentTokenCount → CachedTokens
    Input + Output          → TotalTokens

  Record fields filled:
    Provider/Source = "gemini"          (executor.ProviderGemini constant)
    Alias           = cliproxyusage.RequestedModelAliasFromContext(ctx)
    AuthID/AuthIndex/AuthType = auth.ID/Index/Provider when auth != nil
    RequestedAt/Latency       = startedAt / time.Since(startedAt)
    Failed          = resp.StatusCode >= 400 (or stream err)
    Fail.StatusCode + Fail.Body (truncated)

  Files left unmodified: claude.go, openai.go, codex.go, vertex.go,
  usage_parser.go, util.go, doc.go (explicit constraint).
